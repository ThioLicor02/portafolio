ME PRESENTO

Desde el inicio de mi formación en Ingeniería de Sistemas, he mantenido un enfoque constante en la exploración de tecnologías emergentes y buenas prácticas en el desarrollo de software. Mi objetivo es crear soluciones eficientes, escalables y centradas en el usuario, aplicando conocimientos sólidos en áreas clave como ciberseguridad, redes, análisis de datos y programación. Me caracteriza una mentalidad proactiva hacia el aprendizaje continuo, siempre en busca de oportunidades para aplicar mis habilidades en contextos reales y aportar valor tangible a los proyectos en los que participo.

Mi experiencia se orienta al desarrollo full stack, con dominio tanto del frontend como del backend, lo que me permite participar en todas las etapas del ciclo de desarrollo de software. Disfruto colaborar en entornos multidisciplinarios, aportando una visión técnica y orientada a resultados. Me motiva enfrentar nuevos desafíos que impulsen mi crecimiento profesional y me preparen para contribuir activamente en el dinámico mundo de la tecnología.

Semana 01

Comprendí los conceptos básicos de la web, como tipos de soluciones (sitio, aplicación y sistema web), protocolos (HTTP, TCP/IP, DNS) y roles en el desarrollo (frontend, backend, fullstack). Además, conocí Visual Studio Code como entorno de desarrollo.

RESULTADOS:
Instalé y configuré Visual Studio Code, exploré su interfaz, utilicé extensiones útiles y aprendí atajos de teclado para mejorar mi productividad.

REFLEXIÓN:
Aprendí los fundamentos teóricos y prácticos mediante clases, ejemplos guiados y exploración del entorno de desarrollo. Esto me permitió tener una base sólida para los siguientes temas del curso.

LIBRERÍAS:
  • MDN Web Docs - Web Technology for Developers
  • Visual Studio Code Documentation
  • How the Web Works - MDN
  • TCP/IP and HTTP Fundamentals
  • DNS Explained - Cloudflare Learning

Semana 02

Estructura y etiquetas principales de HTML5 y XML. Árbol DOM, ciclo de vida de una página web, principios de SEO y compatibilidad entre navegadores. Introducción a CSS3: sintaxis, unidades, display, Flexbox, Grid, diseño responsivo

RESULTADOS:
Construcción de una página web usando HTML5 y diseño con CSS3 utilizando Flexbox, diseño fluido y responsivo.

REFLEXIÓN:
Aprendí a estructurar páginas con HTML y a diseñarlas con CSS3 aplicando diseño adaptable. Lo logré practicando con ejemplos y aplicando teoría en el laboratorio.

LIBRERÍAS:
  • HTML5 Specification - W3C
  • MDN HTML Guide
  • Google SEO Starter Guide
  • DOM Introduction - MDN
  • CSS: The Definitive Guide (Online)
  • CSS Grid Guide - CSS Tricks

Semana 03

Uso de Bootstrap para construir interfaces rápidas con componentes predefinidos. Introducción a Tailwind CSS: utilidades para diseño, tipografía, layout, flexbox y componentes de UI.

RESULTADOS:
Desarrollo de un sitio web completo aplicando estilos con Bootstrap y Tailwind.

REFLEXIÓN:
Aprendí a utilizar frameworks CSS para agilizar el desarrollo visual. La práctica guiada me ayudó a comparar estilos tradicionales con utilitarios modernos.

LIBRERÍAS:
  • Bootstrap Documentation
  • Tailwind CSS Documentation
  • Tailwind UI Components
  • Tailwind CSS Cheat Sheet

Semana 04

Fundamentos de JavaScript y TypeScript: variables, tipos de datos, operadores, funciones (flecha, anónimas, closures). Manipulación del DOM

RESULTADOS:
Desarrollo de una animación interactiva utilizando JavaScript, accediendo al DOM

REFLEXIÓN:
Aprendí a controlar el comportamiento dinámico de las páginas con JS y visualicé el potencial de Canvas para animaciones. Lo aprendí mediante teoría aplicada y programación directa.

LIBRERÍAS:
  • JavaScript Guide - MDN
  • JavaScript.info - The Modern JavaScript Tutorial
  • TypeScript Handbook
  • TypeScript Deep Dive (Free Book)
  • JavaScript DOM Manipulation - W3Schools

Semana 05

Introducción a React: estructura basada en componentes, uso de JSX, props y children. Manejo de estilos con distintos métodos (inline, módulos, styled-components).

RESULTADOS:
Creación de componentes en React que reciben y muestran datos usando props y children.

REFLEXIÓN:
Aprendí cómo estructurar interfaces modernas con React. Lo logré experimentando con componentes y pasando datos entre ellos.

LIBRERÍAS:
  • React Official Documentation
  • React Beta Docs (New)
  • Create React App Documentation
  • JSX In Depth - React Docs
  • CSS Modules GitHub

Semana 06

Manejo de eventos, renderizado condicional e iterativo en React. Consumo de APIs con fetch, async/await y Axios. Introducción al enrutamiento y formularios.

RESULTADOS:
Desarrollo de una app en React que muestra datos de una API y responde a eventos del usuario.

REFLEXIÓN:
Aprendí cómo conectar aplicaciones con datos reales usando APIs y a mostrar contenido dinámico. Comprendí mejor el ciclo de vida de los componentes al practicar con código real.

LIBRERÍAS:
  • React Forms Documentation
  • React Router Documentation
  • Axios GitHub
  • Fetch API - MDN
  • Axios Documentation
  • JavaScript Promises - MDN

Semana 07

Uso de hooks en React: useState, useEffect, useContext, useRef, useReducer, useCallback y useMemo. Introducción a hooks personalizados.

RESULTADOS:
Implementación de hooks para manejar estado, efectos secundarios y contexto en una aplicación React.

REFLEXIÓN:
Aprendí a optimizar el rendimiento y la reutilización de lógica en componentes con hooks. La práctica me permitió entender cómo funcionan internamente y cómo simplifican el código.

LIBRERÍAS:
  • React Hooks Documentation
  • React Hooks API Reference
  • Custom Hooks Guide - React Docs
  • React Performance Optimization with Hooks
